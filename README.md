# Scintillating Materials Research Proposal

This repository contains our research proposal on the correlation of material properties with radiation response in emerging scintillator materials, including hydrogels, nanocomposites and CaF₂ crystals.

## Project Overview

This project explores the potential of alternative scintillating materials for radiation detection, medical imaging and radiation therapy applications. The study compares materials such as agarose-based hydrogels, nanocomposites, CaF₂ crystals and plastic scintillators across different thicknesses.

The aim of the project is to evaluate their light output and signal response to understand whether these materials could be used as low-cost and effective radiation detection materials.
